Detailed Description

Core tensor library providing automatic differentiation and memory management.

This library implements a minimal tensor API with separate contiguous storage for data and gradients.

  • ParameterStore manages memory allocation and owns data/grad buffers.
  • Tensor provides a lightweight view into the buffers with shape and offset information.
  • Supports automatic differentiation via a tape-based system.
