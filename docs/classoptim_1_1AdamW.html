Detailed Description

template<typename Scheduler>
class optim::AdamW< Scheduler >

AdamW optimizer with decoupled weight decay.

Template Parameters
SchedulerLearning rate scheduler type

Similar to Adam but applies weight decay before gradient update: p -= lr * λ * p, then Adam update on the decayed gradient.
