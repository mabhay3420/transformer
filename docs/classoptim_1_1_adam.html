Detailed Description

template<typename Scheduler>
class optim::Adam< Scheduler >

Adam optimizer with weight decay.

Template Parameters
SchedulerLearning rate scheduler type

Implements Adam algorithm: m = β1*m + (1-β1)*∇, v = β2*v + (1-β2)*∇² p -= lr * m̂ / (√v̂ + ε), with optional AMSGrad and weight decay.
